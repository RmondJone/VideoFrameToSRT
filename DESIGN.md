# Design System: Video Upload & Processing
**Project ID:** 16700345063590008645

## 1. Visual Theme & Atmosphere

**整体氛围：现代、专业、沉浸式暗色设计**

这是一款面向视频创作者的 AI 字幕处理工具，采用深色模式为主，营造出专业工作室的氛围。界面密度适中，留白充足，不会给人过于拥挤的感觉。双窗格布局让用户可以同时查看视频和字幕编辑，提升工作效率。整体视觉风格简洁现代，强调内容本身而非界面装饰。

---

## 2. Color Palette & Roles

### 主色调 (Custom Color)
- **科技蓝 (#38b9fa)** - 用于主按钮、强调元素、活跃状态指示器。这是整个界面的视觉焦点色，代表 AI 智能处理的科技感。

### 背景色系
- **深邃黑 (#0a0a0a)** - 主要背景色，用于最底层的页面背景
- **炭灰 (#171717)** - 卡片和容器的背景色，提供适度的层次分离
- **深灰 (#262626)** - 输入框、次级容器的背景
- **中灰 (#404040)** - 边框、分隔线的颜色

### 文字色系
- **纯白 (#ffffff)** - 主要标题和重要文字
- **浅灰 (#a3a3a3)** - 次要说明文字
- **暗灰 (#525252)** - 禁用状态、辅助信息

### 功能色系
- **成功绿 (#22c55e)** - 上传成功、完成状态
- **警告橙 (#f97316)** - 处理中、等待状态
- **错误红 (#ef4444)** - 失败状态、删除操作

---

## 3. Typography Rules

**字体：Inter**

- **大标题 (24px-32px):** 使用 Inter Bold (700)，字间距略微收紧，营造专业感
- **中等标题 (18px-20px):** 使用 Inter SemiBold (600)，用于区块标题
- **正文 (14px-16px):** 使用 Inter Regular (400)，舒适的阅读体验
- **辅助文字 (12px):** 使用 Inter Medium (500)，用于标签、状态提示
- **字间距：** 标题使用 -0.02em 的负字间距，正文使用默认间距

## 4. Component Stylings

### 按钮 (Buttons)

**主要按钮 (Primary Button)**
- 形状：圆润胶囊形 (pill-shaped)，8px 圆角
- 背景：科技蓝 (#38b9fa)
- 文字：白色 (#ffffff)，14px Medium
- 悬停：亮度提升 10%，轻微上移阴影效果
- 激活：亮度降低 5%，transform scale(0.98)

**次要按钮 (Secondary Button)**
- 形状：圆润胶囊形，8px 圆角
- 背景：透明
- 边框：1px 中灰 (#404040)
- 文字：浅灰 (#a3a3a3)，14px Medium
- 悬停：边框变亮，背景变为深灰 (#262626)

**图标按钮 (Icon Button)**
- 形状：圆形，40px 直径
- 背景：深灰 (#262626)
- 图标颜色：白色 (#ffffff)
- 悬停：背景变为中灰 (#404040)

### 卡片/容器 (Cards/Containers)

**主要容器**
- 圆角：12px 圆角
- 背景：炭灰 (#171717)
- 边框：1px 透明或极淡的边界
- 阴影：微弱的下沉阴影，营造层级感

**输入框 (Inputs)**
- 形状：8px 圆角
- 背景：深灰 (#262626)
- 边框：1px 中灰 (#404040)，聚焦时变为科技蓝 (#38b9fa)
- 内边距：12px 水平，10px 垂直
- 文字：白色 (#ffffff)

### 进度指示器 (Progress Indicators)

**进度条 (Progress Bar)**
- 高度：4px
- 背景轨道：深灰 (#262626)
- 填充：科技蓝 (#38b9fa)，带有平滑动画
- 圆角：2px 端点

**加载动画 (Spinner)**
- 形状：圆形旋转指示器
- 颜色：科技蓝 (#38b9fa)
- 尺寸：20px-24px

## 5. Layout Principles

### 整体布局

**页面结构**
- 最大内容宽度：1440px，居中显示
- 内边距：24px-32px
- 背景：深邃黑 (#0a0a0a) 铺满整个视口

**双窗格布局 (Dual-Pane)**
- 左侧窗格：视频播放器区域，占比 55%-60%
- 右侧窗格：字幕编辑区域，占比 40%-45%
- 窗格间距：16px
- 可调节的分隔拖拽条

### 间距系统

**间距层级**
- xs: 4px - 紧凑元素间距
- sm: 8px - 关联元素组
- md: 16px - 内容区块内部
- lg: 24px - 主要区块间距
- xl: 32px - 页面边距
- 2xl: 48px - 大区块分隔

### 对齐原则

- 所有内容采用 8px 基准网格对齐
- 文字基线与图标垂直居中
- 按钮组右对齐或左对齐，视场景而定
- 列表项采用左对齐，状态指示器右对齐

## 6. 屏幕设计要点

### Screen 1: Dual-Pane Subtitle Editor (双窗格字幕编辑器)

**左侧 - 视频播放区**
- 16:9 视频播放器容器，深色背景
- 底部控制栏：播放/暂停、进度条、时间显示
- 右上角：设置按钮 (齿轮图标)
- 视频区域可全屏展开

**右侧 - 字幕编辑区**
- 顶部：字幕片段列表（时间轴形式）
- 中部：字幕文本编辑区域
- 底部：AI 分析按钮、导出按钮
- 实时显示当前选中片段的时间范围

### Screen 2: SubtitleAI Dashboard (集成上传仪表盘)

**顶部导航**
- Logo 和项目名称
- 用户头像/设置入口

**主要操作区**
- 大型拖拽上传区域（虚线边框，中心图标+文字提示）
- 支持点击或拖拽上传视频文件
- 上传进度指示（环形进度条）

**配置面板**
- AI 模型选择下拉框（支持 DeepSeek/OpenAI/Minimax/GLM）
- API Key 输入框（密码类型，保护隐私）
- 帧提取间隔设置
- 分析语言选择

**状态显示**
- 当前项目列表（如果已上传视频）
- 每个项目的处理状态标签
